# Geo Dash Mini

A compact recreation of the classic Geometry Dash experience, implemented in Python and adapted for browser-based play.

Geo Dash Mini was originally developed as a desktop game using Python and OpenGL, then ported to run directly in the browser, making it accessible without requiring local installation. The project demonstrates how a real-time game can be built from a single Python codebase and adapted for web deployment.

## Overview

This repository contains:

* The original Python implementation of the game
* A browser-playable version
* Game assets and supporting files
* Deployment-ready web build

The goal of this project was to explore game development concepts such as rendering, collision detection, procedural obstacle generation, game loops, and cross-platform deployment while maintaining a lightweight codebase.

## Running the Project

### Desktop Version

Install the required Python dependencies and execute the main Python file.

```bash
python GeoDash.py
```

### Browser Version

Open the deployed version or serve the repository locally using any static web server.

---

## Why This Project?

Geo Dash Mini was created as a hands-on experiment in game development and deployment. Rather than focusing on large frameworks or engines, the project demonstrates how core game mechanics can be implemented from scratch using a minimal technology stack.

The browser version further showcases how traditional desktop projects can be adapted for wider accessibility and distribution.
